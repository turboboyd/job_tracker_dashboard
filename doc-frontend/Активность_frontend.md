# Frontend — Страница «Активность» (`/dashboard/activity`)

## Файл

`src/pages/DashboardPage/DashboardActivityPage.tsx`

## Маршрут

`/dashboard/activity` → `AppRoutes.DASHBOARD_ACTIVITY`

---

## Что реализовано

### Layout
- Три колонки: фильтр (180px fixed) | лента (flex-1) | статистика (flex-1 max-w-[340px])
- Адаптивный `flex-wrap`

### Фильтр (левая колонка)
- Фильтрация по типам: Всё / Отклики / Интервью / Матчи / Направления поиска / Заметки
- Активный фильтр подсвечивается
- Локальное состояние `useState<FilterKey>`

### Лента активности (центр)
- Группировка событий по дням: «Сегодня» / «Вчера» / DD MMM
- Заголовок группы с разделителем и счётчиком событий
- Иконки событий с цветами по типу (ACT_ICONS)
- Форматирование строки: **кто** действие **цель** (dashed underline на цели)

### Данные (текущий прокси)
- Источник: `useDashboardData()` → `matchesAll`
- `deriveActivity()` — преобразует каждую заявку в событие по `updatedAt` + `status`
- Сортировка по `updatedAt` desc
- `groupByDay()` — группировка по дням

### Статистика (правая колонка)
- Card «За 7 дней»: Всего / Откликов / Интервью / Матчей с прогресс-барами
- Card «Серия»: streak дней подряд + 21 цветной квадрат (визуализация активных дней)

### Шапка страницы
- Breadcrumb: Loopboard / Воркспейс / Активность
- Кнопки «Фильтры» и «Экспорт лога» (UI-заглушки)
